## 🎯 核心业务逻辑

### 角色体系（3类）
- **平台/运营方**（平台超管 + 地区运营）— 收款主体
- **供货商** — 集采供货、库存备货、发货
- **渠道商** — 入口铺设、获取佣金

### 佣金体系
- 平台统一收款
- 按比例给渠道商结算佣金
- 渠道商生成带 ID 二维码，扫码锁定归属
- 周结/月结可配置

### 业务流程
```
用户扫码进入 → 浏览商品 → 加购/购买 → 微信支付
                              ↓
                          供货商接单 → 取票 → 打包发货 → 物流配送
                              ↓
                          用户收货 → 完成
                              ↓
                  （如需）申请退款 → 运营审核 → 微信退款
```
