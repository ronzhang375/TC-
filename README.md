# 💐 鲜花文创商城 - Mono-Repo 整合工程项目

本项目是一个完整的**鲜花衍生品文创商城系统**。为了便于原型展示、交互验证与跨端协作，我们在本地将包含"后台 Java Spring Boot 服务"、"Vue 管理后台系统"、"Uni-App C端小程序"的各分支代码，与专为交互联动的**"静态双端联动模拟器"**进行了整合，采用 Mono-Repo（单一主仓库）架构进行整体版本追踪。

## 🎨 C端小程序视觉升级与排版调优

我们在 `C端小程序模拟器.html` 中进行了深度的视觉升级，体现了高档的法式浪漫美学：
1. **浪漫奶油色彩系统**：主色采用奶油玫瑰粉（`#fb7185`），辅色采用治愈薄荷绿（`#a7f3d0`）。在分类网格图标中融入了局部轻微撞色（浅薄荷绿背景与深绿文字），大幅提升了色彩灵动度。
2. **全站纯色奶油底色**：全站视口背景统一设为极简干净的**暖调奶油白纯色（`#fdfbf7`）**，去除了花哨的水印，保证了商品详情、文本和商品明细的可读性与高级感。
3. **购物车排版置底固定**：修复了结算栏悬空露出段底图的 Bug，将 `app-body` 的 padding-bottom 精确调整为与导航栏等高的 `54px`，使结算栏无缝贴合在 Tabbar 正上方；同时将列表滚动容器的 padding 设为 `110px`，使得最末尾的商品在滑至最底部时能够完全露出来，不被任何功能条遮挡。

## ⚙️ 运营管理后台"系统配置"重构与大 Banner 装修

我们在 `运营管理后台.html` 中对原先堆叠杂乱的"系统配置"页面实施了全面卡片化、Tab 化重排：
1. **选项卡式布局**：将配置分门别类为 `⚙️ 全局参数控制`、`🎨 C端内容装修` 和 `🗺️ 地区运营账号` 三个子 Tab 选项卡，界面清晰，操作流程连贯。
2. **所见即所得装修**：在 `C端内容装修` 下，管理员可以直接输入小程序首页 Banner 的主副标题，并通过图片输入框上传本地图片。通过 `FileReader`，系统自动将图片转为 **Base64** 编码存储在本地缓存中。
3. **双端免刷新实时同步**：运营后台保存 Banner 修改后，利用 HTML5 `storage` 事件总线，另一侧打开的 C 端小程序在**完全无需手动刷新**的情况下，主标题、副标题及大背景图均会自动实现秒级同步刷新渲染！

## 🚀 启动与演示指南

为了直观查看项目的业务流与视觉效果，本工程提供了开箱即用的静态双端联动模拟器，无需配置任何 Java、Database 或 Node 复杂环境即可一秒启动：

1. **双端并排打开**：
   - 用现代浏览器（如 Chrome、Edge）在桌面上并排拖开打开 [C端小程序模拟器.html](C端小程序模拟器.html) 与 [运营管理后台.html](运营管理后台.html)。
2. **重置初始化**：
   - 在运营后台右上角点击 **"🔄 重置演示数据"**，以确保数据管道初始化。
3. **操作测试**：
   - **内容装修**：在运营后台系统设置 -> `C端内容装修` 中修改主副标题、上传或清除自定义背景，并点击保存，观察左侧小程序 Banner 的实时无感变更。
   - **商品上下架**：在运营后台商品列表中点击"下架"或"修改价格"，观察小程序对应的列表和详情页实时刷新。
   - **退款与审核流**：在小程序中提交订单并支付（内置微信支付模态框），然后在订单列表对"待发货"订单点击"申请退款"；随后前往后台的"退款审核"选项，可看到退款申请并可点击同意或拒绝，状态在双端全自动流转！

## 🛠️ 后端服务（lottery-mall）— 技术栈与启动

`lottery-mall/` 子工程为体彩衍生品商城小程序的后端服务，基于 Spring Boot 3.2 构建。

### 技术栈

- **框架**：Spring Boot 3.2
- **ORM**：MyBatis Plus 3.5
- **数据库**：MySQL 8.0
- **缓存**：Redis 7
- **API 文档**：Knife4j (Swagger)
- **构建工具**：Maven

### 环境要求

- JDK 17+
- Maven 3.8+
- MySQL 8.0+
- Redis 7+

### 开发环境启动

1. **创建数据库**
   ```sql
   CREATE DATABASE lottery_mall DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
2. **修改配置**：修改 `lottery-mall/src/main/resources/application-dev.yml` 中的数据库连接信息。
3. **启动服务**：
   ```bash
   cd lottery-mall
   mvn spring-boot:run
   ```
4. **访问 API 文档**：http://localhost:8080/api/doc.html

### Docker 部署

```bash
cd lottery-mall
docker-compose up -d
```
